You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=x&num2=y -- Multiplies 2 integers, x and y
  • /github?query -- Runs query with GitHub's REST API
  • /tellme?st1=...&st2=... -- Takes 2 strings and returns them to user
  • /gimmiecats?cname=...&num=x -- For the first parameter, enter "muffin" or "tangerine". Replace x with either 1 or 2. Get a corrosponding image of one of my cats.
